# @StrizhovFactoryBot — контент-фабрика

Telegram-бот, в одном лице играет 5 ролей (Стратег, Аналитик Рынка,
Сценарист, Верстальщик-ТЗ, Метрист). Помогает Сергею Стрижову собирать
поток контента (Reels, ТГ-посты, карусели Instagram) с автоматическим
анализом конкурентов и использованием реальных кейсов клиентов.

## Стек

- aiogram 3 — Telegram-бот
- DeepSeek API через openai SDK — мозг агентов
- aiosqlite — БД (диалог + кейсы + каналы + контент)
- APScheduler — понедельная авто-планёрка
- httpx + bs4 — парсер публичных TG-каналов
- Railway — хостинг

## Команды

`/pack [тема]` · `/drafts` · `/show N` · `/approve N` · `/channel <user>` ·
`/channels` · `/case` · `/cases` · `/done <метка>` · `/pause <ч>` · `/resume`

Свободный текст — отвечает в стиле бренда МСВ через DeepSeek с памятью диалога.

## Расписание

Понедельник 09:00 МСК — автоматическая сборка пакета на неделю.
Сводка приходит в чат, полные тексты в `/drafts`.
